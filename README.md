team29 - 시대적 변화에 따른 경제 이슈의 관심도 변화

[팀원] 경제 202642136 박한별, 경제 202642175 정혜빈

[프로젝트 소개] KDI 경제교육·정보센터의 뉴스 데이터를 활용해 최근 3년간 발생한 주요 경제 이슈의 시대적 흐름을 확인하고 언론 보도 주제에 미친 영향을 한눈에 파악할 수 있게 한다. 수집된 기사와 뉴스 키워드 데이터를 바탕으로 키워드 감성 분석 및 워드클라우드 시각화를 수행하여 특정 기간 동안 어떤 경제 키워드가 가장 많이 등장했는지 파악하고, 분기별로 경제 이슈가 어떻게 이동했는지 분석한다. 

[실행방법] 1. 파이썬 터미널을 열고 필요한 라이브러리들을 설치한다. 2. streamlit run app_29.py를 입력하여 프로그램을 실행한다. 3. 좌측 사이드바에 년도 및 분기별 선택 필터에서 원하는 기간을 선택하면 KDI 경제교육·정보센터에서 가져온 최근 3개년 경제 뉴스 데이터 샘플이 출력된다.

[사용 라이브러리] streamlit, panda, matplotlib, wordcloud, konlpy
